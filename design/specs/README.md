# Xanthos API Documentation Hub

Documentation generated from the official JV-Link (v4.9.0.1) and JV-Data (4901) specifications under `design/source-docs`.

## Available References
- `version.md`: Published version and last update date of the official specs.
- `methods.md`: JV-Link COM method reference, including syntax, parameters, return codes, and notes.
- `properties.md`: Runtime COM property catalog (types, meanings, default behaviour).
- `error_codes.md`: Consolidated negative/positive return codes with causes and remediation hints.
- `events.md`: JVWatchEvent callback list plus payload format and parameter semantics.
- `records.md`: Field-level layouts for every JV-Data record type.
- `code_tables.md`: Lookup tables for codes used across JV-Data (e.g., racecourses, track types, grades).
- `data_types.md`: Mapping between dataset identifiers and record sets.
- `delivery_schedule.md`: Delivery cadence and update timing for each dataset.
- `special_notes.md`: Raw official commentary and caveats to consider when consuming JV-Data.
- `change_history.md`: Full upstream changelog pulled from the official specification.

> These files are generated by `scripts/generate_api_docs.py` and can be refreshed when the upstream specifications change.
